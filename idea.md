Проект: Интернет-магазин (MVP)

Стек:
- Backend: Medusa v2, TypeScript, PostgreSQL
- Frontend: Next.js, TypeScript
- Auth: Google OAuth, VK ID
- Payments: ЮKassa (Карты, СБП, SberPay)
- Docker Compose для локальной разработки

Функционал:
1. Каталог товаров
   - Список товаров
   - Карточка товара
   - Категории
   - Поиск/фильтрация

2. Корзина
   - Работает без регистрации
   - Сохраняется между сессиями
   - Привязывается к пользователю после входа

3. Авторизация
   - Google OAuth
   - VK ID

4. Checkout
   - Имя
   - Email
   - Телефон (обязателен)
   - Способ доставки
   - Способ оплаты

5. Оплата
   - ЮKassa
   - Карты
   - СБП
   - SberPay
   - Webhook обработки статусов оплаты

6. Заказы
   - Создание после успешной оплаты
   - Отображение в Medusa Admin

Архитектурные правила:
- Не изменять Medusa Core
- API → Workflows → Modules
- Внешние интеграции оформлять отдельными модулями
- KISS, без микросервисов и enterprise-функций

Не входит в MVP:
- 1С
- СДЭК/Boxberry
- Бонусы
- Маркетплейс
- B2B
- Мобильное приложение
- SMS-подтверждение

Результат:
Пользователь может выбрать товар, добавить в корзину без регистрации, войти через Google/VK, указать телефон, оплатить через ЮKassa и получить оформленный заказ.
